## Why I added a blog

I wanted a place on this site for longer writing that does not need to fit inside a single homepage card. The homepage is still the quick overview. The blog is where I can go deeper on projects, research, and things I am learning.

## What I want to write here

- Research notes from current work
- Build logs for projects and experiments
- Short essays on machine learning, interpretability, and engineering
- Lessons that are useful enough to document once instead of repeating later

## How this blog is set up

Each post lives in a markdown file inside `content/blog`. The filename becomes the URL slug, so this post is available at `/blog/welcome-to-my-blog`.

When I add a new markdown file, it automatically appears on the main blog page as another clickable entry.
